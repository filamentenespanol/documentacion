## Introducción

Los paquetes de filamentos consumen un conjunto de componentes centrales cuyo objetivo es proporcionar una base consistente y mantenible para todas las interfaces. Algunos de estos componentes también están disponibles para su uso en sus propias aplicaciones y complementos de Filament.

## Componentes del paquete

Los distintos paquetes del proyecto Filament se pueden utilizar fuera de un panel:

- [Acción](acción)
- [Formulario](formulario)
- [Infolista](infolista)
- [Notificaciones](notificaciones)
- [Esquema](esquema)
- [Tabla](tabla)
- [Widget](widget)
